# Fasterize exercice 


## Install

    $ git clone https://github.com/Pauline-dcs/fasterize
    $ cd fasterize
    
## Server side 

    $ cd server
    $ npm install

## Running the project if you are inside the server 

    $ npm run dev

## Simple build for production

    $ npm build
    
## Test 
 
    $ npm test
